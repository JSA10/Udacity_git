Lesson 1 notes: "The basics of the web and html"

World Wide Web (WWW): A web of html documents

The WWW consists of html documents (Hyper Text Markup Language)
These are connected to each other using links (URLs)
They can contain a variety of content types: e.g. text, images, music or video.


How is the web run?


If we imagine three separate stages in a process, starting with a person on a laptop on the left and ending with giant servers that store all the html coded content previously discussed.
The web is the glue in the middle that enables web browsers on people's laptops to find content stored on these servers.
It does so using html code and url link identifiers but also a technology called "Hyper Text Transfer Protocol" (HTTP).
HTTP forms the connection with servers, allowing individual web browsers to find and translate the files stored on them into content that can be displayed for and used by people.


Diving into HTML


HTML is made up of four elements:

Text content (that you see on a web page)
Markups (code that determines what it looks like)
References to other documents (images / videos)
Links to other pages

Structure: Tags consist of an opening tag "", followed by visible content, closed with a matching second tag ""
Each pair of opening and closing tag can be thought of as an element

Attributes: Tags can have attributes, a common example is the anchor tag, which can be assigned links.
Void Tags: Image tags are examples of void tags. Void tags have no text 'content' so don't need closing tag. Image tags contain two attributes: A URL link to actual image, plus an alternative message in case image doesn't load, or as a marker for the blind.
