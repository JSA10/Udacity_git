Key takeaways from HTML-CSS-DOM video HTML and CSS are both "languages" HTML controls the "structure" of a web page CSS controls the "style" of a page (how it looks). When programmers talk about the "DOM" (Document Object Model) they are talking about the tree-like structure of a page. HTML is the language of the web, with set syntax and rules html uses tags to form elements, to form a tree structure that the browser can display CSS also follows syntax and rules.
